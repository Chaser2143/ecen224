I, Dr. Lundrigan, am piloting a new class sophomore-level class, ECEn 224, and I need your help! I need motivated students that are interested in shaping the sophomore student experience for years to come. The class will be taught full time starting next year (Fall 2023). In the meantime, I would like to get a feel for how students enjoy the content, labs, class pacing, etc. I am looking for about 20 intersted sophmore students to help pilot this class. If you are intersted, contact me at lundrigan@byu.edu.

The class will cover the following topics:
  • C Programming. After taking this class, you will be an advance C programmer.
  • Machine Representation. You will learn how a computer translates a C program into a machine code
  • Computer Organization. Learn how computers are structured to be efficient and flexible.
  • Complex System. The semester project is to build a smart doorbell system. You will gain experience with connecting multiple systems to build something really cool.

Frequently Asked Questions

What credit do I get for taking this class?

You get credit for ECEn 323. If you are an EE student and was not planning on taking ECEn 323, then it can count toward ECEn 220. If you have already taken ECEn 220, then it will count as credit for ECEn 424.

What are the prerequisites for this class?

CS 111 (or CS 142)

What will I learn about in this class?

Here are the learning objectives
  • Binary Representation and Computation. Ability to represent numbers in binary and perform binary operations.
  • C Programming. Ability to program using the C programming language.
  • Machine Representation of Programs. Ability to represent programs in assembly language, translate between C and assembly language, and evaluate security vulnerabilities.
  • Computer Organization. Ability to identify basic computer components, including memory hierarchy, I/O, and central processing unit (CPU), and explain how these components work together to create a computer system.
  • I/O Programming. Ability to build a system that securely interacts with I/O devices and computer networks through C programming.

Is there a book?

Yes, Computer Systems: A Programmer's Perspective 3E.

How do I join the class?

The class is add-code only so you will need to email me (lundrigan@byu.edu) to join the class.
